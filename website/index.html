Advanced Meteor Client Addon

Meteorist enhances your Minecraft experience with powerful automation, visualization, and utility features designed for Meteor Client.

Key Features

smart_toy

Smart Automation

Automate gameplay actions like trading with villagers, crafting, sleeping, interacting with doors, using mud bottles, and controlling boats.

visibility

Enhanced Visualization

Visualize game mechanics with Nerd Vision: see mob spawn ranges, turtle egg zones, spawner areas, and more.

swords

Combat Systems

Dominate PvP with Z Aimbot, Z Kill Aura, Z Auto Totem, and smart targeting that respects your aim direction.

build

Utility Toolkit

Essential tools like auto login, item sucker, sign search, disconnect alerts, double-door opener, and slot click automation.

self_improvement

Self-Maintenance

Automatically heal, feed, repair gear, and sneak at block edges - keeping you safe and efficient without manual input.

code

Scripting & Control

Run complex command sequences with the Instructions module and run custom scripts via Minescript Integration.
